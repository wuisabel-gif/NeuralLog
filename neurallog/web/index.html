Getting Started

NeuralLog can either analyze an existing Discord export file or create one here with a token and channel ID.

Step 1

Export your Discord history

You can either export with DiscordChatExporter yourself, or use the workflow controls below to export a single channel directly into a JSON file for NeuralLog.

Step 2

Understand the token step

If you export with a bot, the bot token is only used during export. If you export with a personal user token, that also happens outside NeuralLog and may violate Discord's Terms of Service. Prefer a bot token when possible.

Step 3

Load the export here

After you have a JSON export, NeuralLog can search it, build timelines, evaluate retrieval quality, and compare embedding backends from this interface.

How to get the export and token

Bot token: open the Discord Developer Portal, choose your application, go to the Bot section, and copy the token there.

User token: some people retrieve it from Discord web request headers, but that is riskier and can violate Discord policy. If you choose to use a user token here, do so carefully and only if you understand that risk.

Channel ID: enable Developer Mode in Discord, then right-click the channel and choose Copy Channel ID if your exporter needs it.

Just want to try it first? Use the sample export and sample evaluation paths already filled into the form, then click Search Export, Timeline, or Evaluate.
